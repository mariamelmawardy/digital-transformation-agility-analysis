# Digital Transformation & Business Agility Analysis

## Overview
This project analyzes the relationship between digital transformation and business agility using survey data from 99 businesses.

## Methodology
- Data cleaning and preprocessing
- Feature creation (Digital Score & Agility Score)
- Correlation analysis
- Linear regression

## Results
- Digital Score: 3.22
- Agility Score: 3.35
- Correlation: 0.89 (very strong positive relationship)
- Regression Impact: 0.78

## Key Insight
Higher digital transformation leads to higher business agility, although other factors also influence agility.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
